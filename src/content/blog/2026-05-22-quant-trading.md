"이 종목이 오를 것 같다"는 직감으로 투자를 해본 적이 있다. 결과는 대체로 좋지 않았다. 감이 아닌 데이터로 판단하고 싶다는 생각이 퀀트 트레이딩에 관심을 갖게 된 계기였다.

## 퀀트 트레이딩이란

퀀트(Quantitative) 트레이딩은 수학적 모델과 통계를 기반으로 매매 전략을 설계하고, 그 전략을 코드로 자동화하는 투자 방식이다.

핵심은 두 가지다.
1. **규칙 기반 의사결정** — 감정이 끼어들 여지가 없다.
2. **백테스팅** — 과거 데이터로 전략을 검증할 수 있다.

## 간단한 이동평균 전략

가장 고전적인 전략 중 하나인 골든크로스/데드크로스를 Python으로 구현해봤다.

```python
import pandas as pd
import yfinance as yf

def moving_average_strategy(ticker: str, short: int = 20, long: int = 60) -> pd.DataFrame:
    df = yf.download(ticker, period="2y")["Close"].to_frame()
    df["MA_short"] = df["Close"].rolling(short).mean()
    df["MA_long"] = df["Close"].rolling(long).mean()

    df["signal"] = 0
    df.loc[df["MA_short"] > df["MA_long"], "signal"] = 1   # 매수
    df.loc[df["MA_short"] < df["MA_long"], "signal"] = -1  # 매도

    return df
```

20일 이동평균선이 60일 이동평균선을 상향 돌파하면 매수, 하향 돌파하면 매도하는 전략이다.

## 백테스팅 결과 해석 시 주의점

| 지표 | 의미 | 주의사항 |
|---|---|---|
| CAGR | 연평균 수익률 | 기간에 따라 크게 달라짐 |
| MDD | 최대 낙폭 | 심리적 견딜 수 있는지 확인 |
| Sharpe Ratio | 위험 대비 수익 | 1.0 이상이면 양호 |
| Win Rate | 승률 | 단독으로는 의미 없음 |

백테스팅에서 좋은 결과가 나왔다고 실전에서도 통한다는 보장은 없다. **과적합(Overfitting)** 이 가장 큰 함정이다. 특정 기간의 데이터에 너무 맞춰진 전략은 미래 데이터에서 무너진다.

## 과적합을 피하는 방법

> "전략을 검증하는 데이터와 최적화하는 데이터를 분리해야 한다."

- **Walk-Forward 분석**: 시간을 앞으로 이동하며 검증
- **아웃 오브 샘플 테스트**: 전략 개발에 쓰지 않은 기간으로 최종 검증
- **파라미터 최소화**: 파라미터가 많을수록 과적합 위험 증가

## 다음 단계

이동평균 전략은 입문용이고, 실제로 쓸 만한 전략을 만들려면 훨씬 깊은 공부가 필요하다. 다음에는 팩터 투자와 포트폴리오 최적화를 다룰 예정이다.

투자는 여전히 어렵다. 하지만 데이터로 접근하면 적어도 "왜 틀렸는지"는 분석할 수 있다.
